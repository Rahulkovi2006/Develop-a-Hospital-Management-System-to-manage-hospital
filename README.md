# Hospital Management System

## Overview
This project is a complete Hospital Management System implemented using Flask and SQLAlchemy.

## Features
- **Patient Management:** Adding, updating, and retrieving patient information.
- **Doctor Management:** Managing doctor profiles and specializations.
- **Appointments:** Scheduling and managing patient appointments.
- **Medical Records:** Storing and retrieving patient medical records.
- **User Authentication** (not implemented): Manage user accounts and roles.
- **Hospital Administration** (not implemented): Tools for hospital administrators.

## Setup Instructions
1. Install required packages: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Access the application at `http://127.0.0.1:5000/`
